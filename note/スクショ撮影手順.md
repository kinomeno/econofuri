# スクショ撮影手順（note記事用）

`note/スクショ/` に以下のファイル名で配置してください。

## 撮影前の準備
1. ブラウザを **デスクトップサイズ**（横幅 1280px以上）にする
2. `http://localhost:8030/` を開く
3. **Ctrl+Shift+R** でハードリロード（古いキャッシュ排除）
4. Windowsキー+Shift+S で範囲選択スクショ → クリップボード → ペイントで貼って保存

---

## ファイル一覧

### 01_トップ画面.png
- 投入前のトップ画面
- 中央大ロゴ「エコノフリ」が見えている状態
- 上から下まで（ヘッダー、ロゴ、ミニ図解、ドロップエリア、3カラム説明、フッターまで）

### 02_ファイル投入後.png
- 「テキストを貼り付けて投入する」で適当な文章を投入
- 結果一覧が表示された状態
- 「生成結果 (1件)」が見えるよう範囲をとる
- ヘッダー（左上の縮小ロゴ＋Ver表示）も入れる

### 03_プレビュー画面.png
- 投入後、結果カードの「表示」ボタンを押す
- 表＋裏 Canvas が横並びで見える状態
- 上部の「裏を再作成」ボタンと濃度ドロップダウンも見えるように
- 濃度は **普通** で

### 04_設定パネル.png
- ヘッダーの「設定」ボタン → 設定 overlay
- 4セクション全部が見える状態
- ダミー濃度・画面モード（無料/有料）・有料解放・その他

### 05_擬態モード_スプレッドシート風.png
- **Esc 2連打**でスプレッドシート風UIに切替
- タイトルバーの「月次集計_2026.xlsx」が見える状態
- できればデータ行（既存の結果）も見える状態が望ましい

### 06_小窓モード.png
- ヘッダーの「小窓表示」 → 小窓パネル
- パネルだけでなく、デスクトップ全体（薄く見える）も含めると「縮んでる感」が伝わる

### 07_狂気モード生成例.png（任意）
- 濃度を「狂気」に変えて再生成
- プレビュー画面で「失敗演出」がいくつも乗っているもの
- 何度かガチャって良い感じのものを採用

## 撮影が終わったら

`note/初回公開記事.md` 内の `![説明](スクショ/XX_xxx.png)` のリンクが
すべて表示されているか確認してください。

note にアップロードするとき：
1. note の編集画面を開く
2. 記事本文を `初回公開記事.md` からコピペ
3. 画像は note 側でアップロードしてリンクを差し替え（noteは外部 file:// 参照不可）
4. 公開URLを差し替える箇所2つ：「公開URL：」「本体：」
