エコノフリ テスト用ファイル

ここにはエコノフリ（http://localhost:8030/）の動作確認用ファイルを置きます。

【ファイル一覧】
  01_短文.txt              — 1ページ収まる短文。基本動作の確認用。
  02_長文_複数ページ.txt   — 改ページが発生する長文。各裏ページが別パターンになるか確認用。
  03_メール本文.txt        — メール風の本文。実用シーンの再現。
  04_画像_横長.png         — 横長サンプル画像。
  05_画像_縦長.jpg         — 縦長サンプル画像。
  06_PDF_1ページ.pdf       — 1ページPDF。
  07_PDF_複数ページ.pdf    — 3ページPDF。

【使い方】
  1. ブラウザでエコノフリを開く
  2. このフォルダ内のファイルをエコノフリにドラッグ＆ドロップ
     （または「クリックして選択」で個別選択、もしくはフォルダごとドロップ）
  3. 生成結果（表裏交互PDF）が一覧に表示される
  4. 「印刷」または「PDFを保存」で確認

【テスト観点】
  ・表面が読めるか
  ・裏面に裏紙ダミーが出ているか
  ・濃度（薄い／普通／濃い／狂気）を切替えて見た目の差が出るか
  ・複数投入でファイルごとに別PDFになっているか
  ・スキップ件数が「○件 スキップ」と表示されるか（対応外ファイルが混ざったとき）
  ・フォルダごとドロップで複数同時投入できるか
